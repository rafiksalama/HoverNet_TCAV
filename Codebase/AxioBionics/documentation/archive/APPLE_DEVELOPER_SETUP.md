# Apple Developer Account Setup Guide for AxioBionics

**Document Version**: 1.0
**Last Updated**: 2025-11-15
**Estimated Time**: 2-3 hours
**Cost**: $99/year

---

## Overview

This guide walks you through setting up your Apple Developer account and configuring all necessary certificates, identifiers, and profiles for AxioBionics iOS development.

**Prerequisites**:
- Apple ID (create at appleid.apple.com if needed)
- Credit card or payment method
- Legal business name and address
- D-U-N-S Number (for organizations)

---

## Step 1: Enroll in Apple Developer Program

### 1.1 Visit Enrollment Page

```bash
open https://developer.apple.com/programs/enroll/
```

### 1.2 Choose Enrollment Type

**Options**:
- **Individual**: $99/year - Simplest, for solo developers
